Skinny Label Similarity Analysis

Semantic Analysis of the Skinny Label Trap: Patent Claims, Use Codes, and Drug Label Indications  |  Target: JAMA  |  PI: Jason Gibbons

Research Questions

What is the semantic relationship between patent claims, Orange Book use codes, and approved label indications — and how has this relationship evolved over time to create conditions where the section viii carve-out is insufficient to protect generic entry?
Claim ↔ Use Code: Do patent claims extend beyond their listed use codes?
Claim ↔ Indication: Do patents cover approved indications or broader territory?
Indication ↔ Use Code: Are use codes listed without approved indication matches?
Temporal: How did GSK v. Teva (2021) change the patent-indication landscape?
Network: How does the patent–use code–indication web densify over a drug's lifecycle?
Policy: Which safe harbors would most effectively address the trap?
Claim ↔ Use Code
Does the patent's legal language match its stated purpose in the Orange Book?
Low sim → patent broader than use code → skinny labeling won't protect generics
Claim ↔ Indication
Does the patent cover what the drug is actually approved for?
Low sim → patent is tangential/strategic, not covering real indications
Indication ↔ Use Code
Are use codes tracking real approved indications?
Low sim → use codes added for things not in the label
Derived Construct: The Skinny Label Trap Score
trap_score = mean(claim↔usecode) − mean(indication↔usecode)
Measures the semantic gap between patent scope and approved indication coverage. High score = patents cover territory beyond approved uses = generic cannot safely carve out.

Full Three-Way Analysis Window
Study Period: Full OB patent panel spans 1985–2024 (39 years, gap only at 1986). However, the three-way analysis requires label text, and the Drugs@FDA API only provides digitized labels back to 1998. Earlier labels exist only as paper records in FDA archives. The claim ↔ use code analysis (two-way, no labels needed) can extend back to 1988. The primary analytic cohort is therefore 1998–2024 for all results requiring label indications.
