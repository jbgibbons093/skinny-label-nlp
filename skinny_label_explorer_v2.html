About & Methods

This tool visualizes the semantic relationship between individual patent claims, Orange Book use codes, and approved label indications for FDA-approved drugs. The scope gap (formerly "scope gap") measures how far patent claims semantically exceed what the drug's label actually covers.

+ Show methods

Three-Way Semantic Similarity

All text (patent claims, use code descriptions, label indications) is embedded with S-PubMedBert-MS-MARCO (768-dim, biomedical domain). Cosine similarity in this shared space measures semantic closeness. Three pairwise similarities are computed:
  • Claim ↔ Use Code: Does the patent's language match the use code description?
  • Indication ↔ Use Code: Does the drug label's approved indication match the use code?
  • Scope Gap: = Claim-UC similarity minus Indication-UC similarity. Positive = patent claims reach beyond label.

Scope Gap (formerly Scope Gap)

scope_gap = mean(max_claim_sim_per_UC) - mean(indication_UC_sim)
For each use code, we find the single best-matching claim (max across all independent + dependent claims). The average of these per-UC maxima is compared to the average indication-UC similarity. A positive scope gap means the patent portfolio semantically covers territory that the label does not.

Similarity Floor

Biomedical embeddings have a floor at ~0.82 cosine similarity — even unrelated texts score this high. The meaningful range is 0.82–1.0. Our threshold of 0.88 corresponds to the 10th percentile of true indication↔use code matches. Edges below 0.88 are considered "unmatched."

Independent vs. Dependent Claims

Independent claims define broad patent scope ("A method of treating..."). Dependent claims narrow to specifics ("...wherein the disease is cardiovascular disease"). Both are embedded individually. 228,661 total claims: 52,982 independent, 175,580 dependent. The dependent claim toggle shows how much they contribute to the scope gap.
